Guide d'utilisation :

Étape 1 : Définir les critères
Ajoutez les critères importants pour votre décision (ex: Prix, Qualité, Délai). Minimum 2 critères.


Étape 2 : Définir les alternatives
Ajoutez les options parmi lesquelles vous devez choisir (ex: Voiture A, Voiture B). Minimum 2 alternatives.


Étape 3 : Noter les alternatives
Pour chaque critère, notez chaque alternative de 1 à 10 (10 = excellent, 1 = mauvais).


Étape 4 : Comparer les critères
Remplissez la matrice de comparaison par paires en utilisant l'échelle de Saaty :

1 = Également important

3 = Modérément plus important

5 = Fortement plus important

7 = Très fortement plus important

9 = Extrêmement plus important

Étape 5 : Obtenir la décision
Cliquez sur "Calculer". Si la matrice est cohérente (CR < 0.1), la meilleure alternative s'affiche. Sinon, ajustez vos comparaisons.

 Exemple d'utilisation
Achat d'une voiture :

Critères : Prix (60%), Confort (30%), Consommation (10%)

Alternatives : Toyota (52%), Honda (32%), Nissan (16%)
